# template
This is a base template for hooman who what a short cut to host their app.

## Installation
1. npm install
2. npm install -g gulp-cli

## Structures 
* Add your application
--app
    --css
    --js
    --temp
        --style
            --style.css 
    --networkModule

* Add your auto tasks 
--gulp 
    --tasks
        --jshint.js
        --styles.js
        --watch.js
--gulpfile.js

* Add your dependencies 
--package.json

## Shortcuts
For Chrome
* Open whatever panel you used last	
Command+Option+I	F12 or Control+Shift+I

* Open the Console panel	
Command+Option+J	Control+Shift+J

* Open the Elements panel	
Command+Option+C	Control+Shift+C

For vsCode
* format
Command+Option+F
